Privacidad y API

Configura Gemini

Tu API Key no se incluye en el código fuente. Se guarda únicamente en el almacenamiento local de este navegador bajo la clave nutriscan_api_key. La app consulta los modelos habilitados para tu API Key e intenta usar primero Gemini 2.5 Flash; si no aparece disponible en tu proyecto, baja automáticamente a otro modelo Flash compatible.

La app funciona sin servidor propio: la llamada a Gemini sale desde tu navegador.

Asistente inteligente

Configura NVIDIA NIM

Tu API Key de NVIDIA se guarda únicamente en el almacenamiento local bajo la clave nutriscan_nvidia_api_key. Esto habilita el Asistente nutricional con chat, generador de recetas y análisis avanzado usando modelos como Llama 3 o Mistral.

Obtén tu API Key gratis en build.nvidia.com. La llamada al Asistente sale desde tu navegador.
