Privacidad y API

Configura Gemini

Tu API Key no se incluye en el código fuente. Se guarda únicamente en el almacenamiento local de este navegador bajo la clave nutriscan_api_key. La app intenta usar primero Gemini 3.5 Flash y, si tu cuenta aún no lo tiene habilitado, baja automáticamente a modelos Flash compatibles.

La app funciona sin servidor propio: la llamada a Gemini sale desde tu navegador.
